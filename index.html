해낼 고3, 1년의 흐름을 설계하다

수능까지 1년. 흐름만 설계되면, 성적은 그 흐름을 따라옵니다.

연간·월간·주간 루틴과 피드백 시스템으로 “흐름이 끊기지 않는 고3”을 만드는 프로젝트입니다.

Year Roadmap

1년 성적 향상 로드맵

고3 한 해를 네 구간으로 나누어, 시기마다 해야 할 일을 분명하게 정해 둡니다.

1~2월 · 기본기 완성기 취약 과목·단원 집중 보완
전 범위 핵심 개념 총정리
3~6월 · 심화·실력 성장기 심화 개념 + 고난도 문제 훈련
풀이 체력·집중력 증가 구간
7~8월 · 실전 대비기 실전 시즌 : 매회 분석 + 약점 보완
시간 배분·풀이 루틴 확립
9~11월 · 최종 점검기 파이널 전략 수립
실전 컨디션 관리 · 점수 유지·상승 구간

이 로드맵을 현실로 만드는 연간 전략

  1. 학습 목표 및 방향 설정 – 현재 위치 진단, 목표 대학·등급 설정
  2. 연간 학습 구조 설계 – 수능/내신/논술 비중 조정, 과목별 플랜 구성
  3. 점검 & 피드백 – 주간/월간 점검과 루틴 수정으로 흐름 유지

※ 모집 대상: 인천 검단 지역 고3만 모집합니다. 검단 지역 고2는 본 프로젝트에서 모집하지 않습니다.

Flow System

해낼 고3, 흐름을 일상으로 정착시키다

월간 학습 계획

  • 이번 달 반드시 완성해야 할 핵심 개념·유형 범위 설정
  • 집중 학습 과목 선정 및 교재·자료 확정
  • 1~4주차 세부 학습 내용과 목표를 주차별로 배치

주간 학습 계획

  • 이번 주 달성해야 할 구체적 목표 1~2개 설정
  • 우선순위 과목 지정, 요일별 루틴 구성
  • 월~토(또는 일) 단위로 학습량·시간 배분

평가 & 피드백 시스템

  • 주간 테스트 및 일일 점검으로 학습 성취도 확인
  • 오답노트 분석 → 다음 계획에 바로 반영
  • 주간 상담을 통해 학습 방법·루틴·취약점을 지속 보정

우독학원은 학생의 수준과 목표에 맞춰 연간·월간·주간 계획을 설계하고, 상담과 피드백으로 흐름이 끊기지 않도록 관리합니다.

Process

『해낼 고3』 운영 프로세스

학생이 등록한 순간부터 수능 직전까지, 이렇게 관리합니다.

  1. 학생 등록 후 레벨진단으로 주력·취약·치명적 결함 파트 구분
  2. 수능 / 내신 / 논술 3트랙으로 1차 전략 수립
  3. 학생별 주간 루틴표 작성 후, 매주 루틴을 조정하며 진행
  4. 학생이 매주 학습 보고서를 제출하고, 강사가 이를 기반으로 피드백
  5. 월말 성적·학습량·멘탈 종합 점검, 학부모 상담으로 공유
  6. 성적·멘탈 변화에 따라 전략 재배치, 수능 직전 집중관리 모드 전환
자세한 운영 프로세스 보기

1) 학생 등록 → 레벨진단 (주력·취약 파악)

✔ 무엇을 검사하냐?

  • 수학 – 전 범위 개념 체크리스트 문항 테스트, 최근 모의고사 분석(오답 패턴·찍은 문제·시간 배분), 미적/기하/확통 선택 여부
  • 국어 – 비문학 독해 속도 / 문학 취약 유형, 최근 학평 지문 3개 셋업해서 풀이하게 해보기
  • 영어 – 문법/어휘/독해 각각 진단
  • 생활 루틴 – 기상시간, 취침시간, 등교시간, 하루 공부시간, 핸드폰 사용량(중요)
  • 멘탈/성향 – 회피형/실행형, 멘붕옵션(작년 수험생 애들 보며 느끼는 공통 요소 체크)

✔ 출력되는 결과물 – “주력 파트 / 취약 파트 / 치명적 결함 파트”로 구분

  • 미적: 개념은 강한데 기출 형식 취약
  • 국어: 비문학 속도 280 → 210으로 줄여야 함
  • 영어: 문법 구멍 7개
  • 생활: 수면 불안정(밤 2시 취침)

→ 이 자료가 1차 전략 수립의 재료가 됩니다.

2) 1차 전략 수립 → 수능/내신/논술 구성

✔ 크게 3트랙 전략을 만든다

  • ① 수능 트랙
    · 학생의 현재 점수 기준 설정 (예: 수학 3→2 / 국어 4→2 / 영어 2→1 유지)
    · 수능까지 남은 기간 역산 → 월별 목표량 배치
    · 취약 단원별 보정 전략 입력 (지수·로그 취약 → 유사 20문항 / 수열 극약처방 → 추론형 5일 반복)
    · 모의고사 회차별 전략, 목표 점수, 시간 배분 방식 지정
  • ② 내신 트랙
    · 1학기 / 2학기 지필 대비
    · 학교별 기출 스타일 분석
    · 수행평가·단원평가 대비 일정 입력
    · 고3이라도 학교에서 요구하는 것(성적 유지 + 결석·지각 관리) 포함
  • ③ 논술 트랙
    · 인문논술, 수리논술 대학 희망 여부 확인
    · 기초논술 개념 잡기
    · 과학논술은 의대만 운영
    · 수리논술은 2월부터 주 1회 운영

3) 주간 루틴 설계 → 시작

✔ 학생별로 주간 루틴표 생성

예시:

  • 이번 주에 완성할 개념
    • 국어: 중세국어 문법 완성
    • 수학: 수열의 극한 개념 완성, 기출 풀이
    • 영어: 41~45번과 친해지기
  • 월 – 수학: 수열 40분 + 기출 20문제 / 국어: 비문학 2지문 / 영어: 단어 80개 + 구문 3개
  • 화 – 수학: 미적 함수 개형 50분 / 국어: 현대소설 …

✔ 핵심은 3가지

  • 과목 간 비중 조절: 주력 과목 vs 약한 과목 배치
  • 시간 단위 블록 스케줄링이 아닌 “오늘 할 일”에 집중
  • 현실적인 분량 지정 (학생이 80~90%는 해낼 수 있는 양) – 이 루틴은 매주 수정 가능

4) 매주 학습 보고서 제출

✔ 제출 내용

  • 이번 주 공부 총량(시간·양)
  • 수학 / 국어 / 영어 학습·과제 수행률
  • 모의고사 오답 분석
  • 컨디션 상태(수면/식사/스트레스)
  • 핸드폰 사용량
  • 한 줄 소감(의무)

✔ 제출 방식 – 학원 자체 양식에 매주 작성, 상담 시 사용

✔ 강사는 무엇을 보냐?

  • 공부량이 늘었는지
  • 루틴이 지켜졌는지
  • 성적에 반영되고 있는지
  • 멘탈 붕괴 신호가 있는지(중요)

5) 강사 피드백 반영 → 보완계획 수정

✔ 피드백 구성

  • 전략 피드백 – “수학은 개념보다 기출이 더 필요함. 이번 주 기출 30→45로 올리자.”
  • 멘탈/생활 피드백 – “새벽 2시 취침은 절대 안 됨. 1시 전에 자고, 폰은 11시까지 끊자.”
  • 과목별 보정 피드백 – “국어 문학이 약하니까 이번 주는 비문학 2 → 문학 1로 비중 바꾸자.”

✔ 피드백 반영 후 각 강사에게 보고 → 불이행 이유 파악 → 루틴 수정 → 목표치 재설정 → 다음 주 학습표 생성

6) 월말 종합 점검 (성적·학습량·멘탈)

✔ 점검 포인트 5가지

  • 모의고사 점수 변화
  • 취약 단원 해소 여부
  • 학습량 증가폭
  • 생활 루틴 안정성
  • 멘탈 안정성 체크

✔ 출력물 – 월간 성적 레포트, 과목별 성장 그래프, 루틴 그래프, 다음 달 전략 재설정

※ 이 출력물은 우독 내부 브랜드 보고서 디자인으로 제작 예정.

7) 학부모 상담 (월 1회)

✔ 상담 구조

  • 한 달간의 성적 및 변화 요약
  • 주요 취약점 공유
  • 다음 달 계획 프리뷰
  • 학생의 멘탈/습관 변화 설명
  • 필요한 협조 요청 (폰 사용 제한, 수면 지도, 생활 루틴 통제 등)

✔ 키 메시지 – “한 달 전과 비교했을 때 얼마나 ‘해낸’ 게 있는지”를 숫자 + 그래프로 보여줌.

8) 대비 전략 재배치 → 성장 유지

✔ 적용 시점

  • 매달
  • 멘탈 무너지는 기간(6~7월, 9월)
  • 모의고사 성적 급변 시

✔ 재배치 요소

  • 과목 비중 조정
  • 기출 → 실전 모드 전환
  • 논술 비중 조절
  • 쉬는 시간·루틴 재편
  • 숙제량 리밸런스

9) 수능 직전 집중관리 모드 전환

✔ 시점 – 9월 모평 이후 → 11월 수능 직전 3주까지

✔ 모드 구성

  • EBS 연계 체크 총정리
  • 실전 모의 10회 이상 분석
  • 오답 패턴 총정리, 취약 유형 전부 갈아엎기
  • 멘탈 안정 루틴(수면·식사·운동)
  • 시험장 시뮬레이션
  • 실전 대비 파트 3가지 – 시간 운영 / 문제 배치 전략 / 위기 발생 대처 전략

Counseling

주간 상담 & 월간 리포트 시스템

  • 주간 성적 추이를 보며 과목별 공부 시간·난이도를 조절해 실제 성적 향상을 목표로 합니다.

7/22(월) – 최근 2주 학습 흐름을 점검하고, 국어 독서에서 느끼는 난도 상승과 수학 고난도 풀이 체력 부족을 확인했습니다. 국어는 지문 구조화·선택지 검증 루틴을 다시 잡아주고, 수학은 고난도 15문제/일 풀이로 목표를 정했습니다.

7/23(화) – 국어 중심 상담. 선택지 판단 속도 문제를 확인하고, 문단 대표문장 표시와 3분 컷 훈련 세트를 추가했습니다. 이번 달 독서·문학 목표를 다시 정리하며, 국어 성적이 반등 가능한 지점임을 설명했습니다.

7/24(수) – 수학 상담. 풀이를 길게 가져가며 시간 손해가 나는 습관을 조정하고, 개념→유형→고난도 3단계 루틴을 적용하도록 했습니다. 다음 주부터는 영어 어법, 국어 선택지 훈련, 수학 실전세트 비중을 조정하기로 했습니다.

※ 해낼 고3 프로젝트는 수능 최저 등급을 달성하도록 돕기 위해, 끝까지 책임감 있게 관리하는 것을 목표로 합니다.
